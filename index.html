Some projects that have been created include a bookmark by printing press, an infographic, a video commercial, photo editing portfolio. These projects can be found throughout this webpage.

I learned how to do a basic code to create a banner, insert images, navigation, text, titles, authorship, and external hyperlink. There is several different software to code, but we used sublime text. This project was especially important to keep organized and pay attention to detail.
